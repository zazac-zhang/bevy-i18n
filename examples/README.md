# Examples

Run with `cargo run --example <name>`.

| Example | Description | Controls |
|---------|-------------|----------|
| `basic` | Minimal setup: plugin, locales, translated text | — |
| `locale_switch` | Press Space to toggle EN/ZH | Space |
| `advanced_features` | All features: plural, context, TVar, formatting | ↑↓ count, ←→ locale, ± score |
| `custom_component` | `#[derive(I18n)]` for custom components | Space |
